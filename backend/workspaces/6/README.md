# 🦌 GladMe Deer Identification System

This project implements a sophisticated computer vision pipeline to identify and classify specific species of deer from photographic inputs.

The system is designed using a **Microservice/Pipeline Architecture**, where complex tasks (Detection, Cropping, Classification) are separated into independent, modular Python services for enhanced scalability and maintainability.
