FIX #1: JWT Token Timestamp Validation
────────────────────────────────────────
  File:     V5/backend/auth.py (lines 26-34)
  Problem:  Tokens weren't validating; API returned "Invalid or expired token"
  Cause:    JWT claims using datetime objects instead of Unix timestamps
  Solution: Convert to int(datetime.timestamp())
  Status:   ✅ FIXED

  Before:  "exp": expire  # datetime object
  After:   "exp": int(expire.timestamp())  # Unix timestamp


FIX #2: React Hook Dependencies
────────────────────────────────────────
  File:     V5/frontend/src/App.jsx (lines 94-110)
  Problem:  "Something went wrong" error on page load
  Cause:    initApp function not in useEffect dependency array
  Solution: Wrap initApp in useCallback with proper dependencies
  Status:   ✅ FIXED

  Before:  useEffect(() => { initApp(); }, [])
